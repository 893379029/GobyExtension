## 快速开始

1.  安装

2.  开始扫描, 发现Java反序列化漏洞

3.  点击插件按钮, 进入插件弹窗进行配置, 可使用默认配置

4.  生成并回填反序列化数据, 点击验证

5.  验证成功后可使用 ShellHub 插件进行管理
